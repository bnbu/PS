1. Fractional Knapsack : 짐을 쪼갤 수 있음 -> Greedy 사용
2. 0-1 Knapsack : 짐을 쪼갤 수 없음 -> DP를 사용

여기서, 0-1 Knapsack은 다시, 물건의 개수에 따라 또 분류가 가능하다.

2-1 0-1 Knapsack : 각 물건은 최대 1개씩만
    => 물건을 가져가지 않는 경우 / 물건을 가져가는 경우 로 구분지어서 진행

2-1 Bounded Knapsack : 물건마다 최대 개수가 정해져 있음
    => 물건을 가져가지 않는 경우 / 물건을 가져간다면, 몇개 가져가는지를 구분지어 진행

2-3 Unbounded Knapsack : 물건의 최대개수가 정해져 있지 않음 (즉 무제한)
    => 안고르는 경우는 최대 후보로 있을 수 없으므로 모든 경우에 수에 대해 결정
